# 💳 Financial Fraud Detection System

A Machine Learning based system to detect fraudulent financial transactions.
This project uses data preprocessing, feature engineering, model training, real-time prediction, alert system, and a dashboard for visualization.

---

## 📁 Project Structure

financial_fraud_detection/
│
├── alert_system/
├── dashboard/
├── data/
├── database/
├── feature_engineering/
├── models/
├── preprocessing/
├── streaming/
├── visualization/
└── main.py

---

## ▶️ How to Run the Project

1. Install requirements
pip install -r requirements.txt

2. Run data preprocessing
python preprocessing/data_preprocessing.py

3. Train the model
python models/train_model.py

4. Start the system
python main.py

5. Open dashboard
python dashboard/app.py

- Data Cleaning and Feature Engineering
